# 📢 Twitch チャット読み上げくん（スマホ対応版）

スマホのブラウザで開くだけで、Twitchのチャットを「ユーザー名 ＠ コメント」の形式で自動読み上げするツールです。
プログラミングの知識がなくても、5分で自分専用のページが作れます！

## 🚀 導入手順（初心者向け）

### 1. 魔法のコードをコピーする
このリポジトリにある `index.html` の中身をすべてコピーしてください。

### 2. 自分の ID に書き換える
コードの **29行目あたり** にある以下の部分を探して、`' '` の中をご自身の Twitch ID（ユーザー名）に書き換えてください。

// 【ここを自分の名前に変えるだけ！】
const TWITCH_ID = 'あなたのTwitchID';
