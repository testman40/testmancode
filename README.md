# 📢 Twitch チャット読み上げくん（スマホ対応版）

スマホのブラウザで開くだけで、Twitchのチャットを「ユーザー名 ＠ コメント」の形式で自動読み上げするツールです。
プログラミングの知識がなくても、5分で自分専用のページが作れます！

## 🚀 導入手順（初心者向け）

### 1. 魔法のコードをコピーする
このリポジトリにある `index.html` の中身をすべてコピーしてください。

### 2. 自分の ID に書き換える
コードの **29行目あたり** にある以下の部分を探して、`' '` の中をご自身の Twitch ID（ユーザー名）に書き換えてください。

### 3. ブラウザを開いてURLをコピーします
`https://testman40.github.io/testmancode/`をコピーしてページが開けばあとは接続を開始するボタンを押すだけです。配信終了には
接続を終了するを押してください。

## 🆙 最近のアップデート情報

さらに便利に、安定して使えるように以下の機能を追加しました！

- **【NEW】ログ保存機能（永続化）**: 
  - ブラウザを閉じたりリロードしたりしても、過去のコメントログが消えなくなりました。
- **【NEW】未読チェックボタン**: 
  - 接続が不安定になった時や、少し席を外した後にボタンを押すと、最新のコメントを再取得して読み上げます。
- **【NEW】時刻表示**: 
  - 各コメントの横に受信時刻 `[12:34]` を表示。いつ届いたコメントか一目でわかります。
- **【NEW】ラグ防止・爆速モード**: 
  - コメントが溜まっても最新の会話に追いつけるよう、読み上げ処理を最適化しました。
- **【NEW】ログ消去ボタン**: 
  - 溜まったログをワンタップでリセットして動作を軽くできます。
